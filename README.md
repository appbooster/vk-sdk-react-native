# vk-sdk-react-native

VK SDK for React Native

## Installation

```sh
npm install vk-sdk-react-native
```

## Usage

```js
import VkSdkReactNative from "vk-sdk-react-native";

// ...

const result = await VkSdkReactNative.multiply(3, 7);
```
